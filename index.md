		<h2 class="mainheading">About the server</h2>
		<div id="serverinfo">
			<article class="post">
				<p class="administration">The Total Freedom server is a free form sandbox server. Here at Total Freedom, all players are given creative mode plus OPs privileges provided they abide by our rules. Unlike most multiplayer servers, we like to allow maximum freedom here, whether it's using world edit commands, plugin essentials commands, mods, or logging in without a paid/premium account.  We also have a free voice server using Mumble; please check our our <a href="https://web.archive.org/web/20130711051648mp_/http://totalfreedom.boards.net">forums</a> for more information.  Total Freedom does not mean Total Anarchy; if you attempt to disrupt or grief the server, sanctions to include temporary or permanent banning are in play.</p>
				<p class="administration">All players are equal with ops and plugin privileges. However, regular players can be accorded admin status by positively contributing to the server. Positive contributions include producing or providing excellent world maps, making fantastic builds that get saved for use in our sandbox, assisting the server admins in dealing with griefers, and/or cleaning up their messes (e.g. spawn destruction), assisting new players, keeping the server free of lag and so forth.  If you have specialized skills such as plugin coding, please discuss with the appropriate executive board member or owner.  The list of senior admins and super admins are posted on this page and can be viewed in-game by using the command saconfig list.  Senior admins noted as honorary are veteran senior admins that are no longer required to be as active as the regular senior admins but are recognized for their long-standing service to the server. </p>
				<p class="administration" id="admininfo">Super admins are below senior admins &amp; have various capabilities such as being able to OP themselves and overseeing the server in-game.  If 	you wish to be a super admin, please fill out the application on our forum as listed above.  We also have a specialized technical position known as Telnet admins that allow for off-server oversight- all senior admins have Telnet capability as well as designated super admins.  Senior Admins are in-charge of selecting and supervising Super Admins.</p><p></p>
				<p class="administration">Admins should give OPs to all new members (anybody may OP another player) and instruct new players to read the rules and this website. They can also deop griefers (first choice for minor infractions) and temporarily ban griefers that cause major world damage and other serious rule violations.  Admins may use an [Admin] nick title tag if they so wish provided they use their normal nick as listed above but must use their minecraft name when doing admin commands.  If you wish to be a super admin or to know more detailed information about admin duties, please apply on the <a href="https://web.archive.org/web/20130711051648mp_/http://totalfreedom.boards.net">forums</a> and do not NOT publicly nag for it on the server. If you're not yet qualified to be an admin, you can become a crew member (admin assistant) and wear the crew tag if appointed by an admin - see the forums for more info</p>
				<p class="administration">You may contact the server owner via Twitter at @totalfreedommc to pass critical information such as server crashes.  For ban appeals or other non-critical issues, please use our <a href="https://web.archive.org/web/20130711051648mp_/http://totalfreedom.boards.net">forums</a>.</p>
				<h2 id="rules" class="rules">Server Rules</h2>
				<h3 class="rules">No Griefing!</h3>
				<p class="rules">Do not grief the server. There are many way to do that, but the most common are to intentionally crash the server, lag the server, destroy or damage the spawn, destroy or damage other player's builds, erase warp points, PVP without agreement of other players, chat spamming, and so forth. Use your best judgement, and remember that other people are on here to have fun also.</p>
				<p class="rules">While we do strive to give our users Total Freedom, there are some limitations that we have been forced to put in place to prevent the worst of griefing. We are sorry for these measures, and if you need help building something legitimate but can't because of these limitations, please contact an admin.</p>
				<p class="rules">Do note that actions such as "nuking" (removing and placing blocks rapidly via a client side hack) or freecam building/destroying will be automaticlly detected and you will be removed from the server. Additionally, some blocks, when used with worldedit, are prohibited and will result in automatic de-opping for evaulation by an admin. You should get permission from an admin before using "high grief" potential blocks like water and lava.</p>
				<h3 class="rules">No Impersonation</h3>
				<p class="rules">Do not pose as the server owner (markbyron) and do not pose as an admin or one of the other regular players that use Total Freedom. By posing, that means using a program such as MinecraftSP or otherwise changing your nick. Posing to change your title or to pose as somebody "famous" (like Notch) is okay provided you do not use it grief the server.  You may not use nicks or tags that imply you have an official position on the server.  This includes any nick or tag including the words admin, senior, owner, telnet, builder, architect, coder, dev, programmer, mod, security, or any nick or tag that the owner or a senior admin believes is an attempt to pose.  In such cases, the offender will be de-nicked and warned.  You may however use the tag [helper] if you're interested in becoming an admin and wish to assist admins.  In all other cases, nicks and tags must not violate other rules such as respect for other players.  For example, using the nick markbyronSucks will not be smiled upon.  Last but not least, do not use spinning nicks, line-thru nicks, underline, bold nicks, black nicks (too hard to see), super long nicks, all-upper case, blank/hidden/garbage character nicks, or any form of nick that's designed to spam or annoy other players. If you login without premium (offline mode), a login name that is not valid by Mojang standards is disallowed (containing characters that are not numbers, letters, or an underscore to include spaces.)</p>
				<h3 class="rules">Respect Privacy</h3>
				<p class="rules">Respect the "Total Freedom" of other players. Do not be a pest or a nuisance to others, and respect their privacy. Do not "help" with builds if people do not want you to, and do not teleport someone without their permission. It is best to use <span class="command">/tpahere</span> to request that someone to teleport to you, rather than forcefully teleporting them.  There are numerous way to be a nuisance and we don't detail all of them here but you'll learn them with time.  Please note that privacy of communication is not yet offered on this server; all private messages / mail are logged plus all members have have access to socialspy.  Use external services such as Skype or Twitter (Direct Message) if you require private communication with other members.</p>
				<h3 class="rules">Respect other players and admins</h3>
				<p class="rules">Do not engage in personal attacks on other players/admins and take all personal disputes off the server.  If you don't like a particular player, feel more than free to use the ignore command on them but personal wars on the server won't be tolerated regardless of who started it and temporary bans will result.  If you're coming on the server to troll other players, you'll end up getting trolled yourself (we love to play with trolls).   Similarly, threats to DOX, threats to commit illegal acts  (e.g. DDOS), graphic sexual discussion or RP, linking to porn or scam sites, racist attacks, and other forms of behavior designed to offend, intimidate or harass will not be tolerated.  This rule applies to admins to players as well, and admins that abuse their authority to engage in personal attacks will be sanctioned or removed.  Respect between admins is essential on the server and on the forums.  Personal disputes, insults, criticisms, and complaints between admins are to be handled off-server and in-private (Forum PM, Twitter DM, Skype).  The rule also applies to forums (including admin sections) or anywhere that the dispute might be viewed by the general public (e.g. public Twitter).  Ex-Admins or suspended admins who continue on the server as regular OPs or a regular on the forum will be held to same standard of conduct between admins and failure to do so will result in temporary or permanent bans (server and forums).  Excuses such as "I didn't start it" will not be accepted - anybody involved in a public 'admin war' is equally guilty</p>
				<h3 class="rules">Adhere to Authority</h3>
				<p class="rules">Do not override the actions of the server owner (markbyron) or other admins. For example, if an admin de-ops or bans a player for griefing, you may not re-op or pardon them without permission.  Super admins will follow the directions of senior admins with respect to any admin matter unless it violates server rules in which case a PM will be sent by the super admin to the owner.</p>
				<h3 class="rules">Consequences</h3>
				<p class="rules">Violation of these rules may result in a temporary de-op, temporary IP ban, or other disciplinary action. The length of a ban will be determined by the server owner based on the severity of the violation.</p>
				<p class="rules">If you feel you have been banned, muted, or had an administrative action applied to you unfairly, please contact the server owner via Twitter at @totalfreedommc</p>
				<h3 class="rules">Perm Ban Release Rules</h3>
				<p class="rules">Except for those who were improperly placed on the perm ban list, those who get released from perm ban under any other condition are subject to special a special rule and restriction for up to one year after release.  The special rule is that you will be be re-added to the perm ban list (including any alts and previously used IPs) for any offense that would normally get a temporary ban, and the special restriction is not being able to apply for super admin for a minimum of 90 days after release from perm ban or 365 days if applying for Telnet or Senior.  The special rule and restriction will be in place for 365 days following release from perm ban or earlier if the owner and all senior admins agree.</p>
				<h2 class="help">Server Help</h2>
				<h3 class="help">Saving and Loading Builds With WorldEdit</h3>
				<p class="help">Due to the immense and immediate world changing capabilities of the plugins this server runs, and the influence of frequent griefing, the server is frequently reset and restored to a default saved map. Except as arranged with the server owner (markbyron), <span class="bold">all builds, experiments, and user inventories are temporary unless you save them using WorldEdit</span>.</p>
				<p class="help">To save your builds (i.e. create a WorldEdit schematic), first use the <span class="command">//wand</span> tool to select the bounds of your build (i.e. the opposite corners of your build, see the <a href="https://web.archive.org/web/20130711051648mp_/http://wiki.sk89q.com/wiki/WorldEdit/Quick_start">WorldEdit Quickstart Guide</a> for help.) Then, type <span class="command">//copy</span> to place your build on the clipboard, and type <span class="command">//schem save nameofbuild</span> to save your build (replacing <span class="command">nameofbuild</span> with a unique name of your own).</p>
				<p class="help">To load a schematic back into the world, do <span class="command">//schem load nameofbuild</span>, and then <span class="command">//paste</span>.</p>
				<h3 class="help">WorldEdit Cautions</h3>
				<p class="help">When using the WorldEdit plugin, avoid selecting a radius greater than 50 or an area greater than 25,000 blocks. Selecting overly large numbers will typically lag and 	crash the server (which of course is a violation of the rules above). If you make a World Edit that accidentally causes destruction to the spawn or other player builds, you should immediately say <span class="command">//undo</span>. While accidents can happen, be responsible, and if you're just learning WorldEdit, test the commands in remote part of the map far away from any other builds.</p>
				<h3 class="help">Getting OP'd</h3>
				<p class="help">When you join the server, in order to use the full set of commands you must be "opped". Simply request an admin give you OP permissions.  Without OPs, you are default to creative mode and may obtain any inventory item.</p>
				<h3 class="help">Vanilla Server</h3>
				<p class="help">If the CraftBukkit (the software this server runs on) is unavailable due to a recent Minecraft software change, we will be forced to use the standard Minecraft server software. As the standard server software is extremely limited, we can only give ops to existing admins and other players will be given creative mode by default.</p>
				<p class="help">A limited set of commands work with the Vanilla server, for more information see <a href="https://web.archive.org/web/20130711051648mp_/http://www.minecraftwiki.net/wiki/SMP_Server_commands">Vanilla Server Commands</a>.</p>
				<h3 class="help">Flatlands</h3>
				<p class="help">This server features an additional flat-land building world which is very useful for building intricate creations. You can access this world by typing <span class="command">/flatlands</span>, and you can get back to the main world by typing <span class="command">/flatlands</span> again.</p>
				<h3 class="help">Nether and "The End"</h3>
				<p class="help">You can use standard portals to access the Nether but the End world has been temporarily disabled due to crash exploitation</p>
				<h3 class="help">Gamemodes</h3>
				<p class="help">You are free to either use creative or survival mode on the server. For creative, type <span class="command">/creative</span>. For survival, type <span class="command">/survival</span>.</p>
				<h3 class="help">Other Plugins</h3>
				<p class="help">As Minecraft is continually updated, there is no guarantee that the mods and plugins will continue to function or that the developers will fix them when Minecraft is updated. We will do our best to update or replace mods and plugins as needed. If you have suggestions, please send a message to MarkByron.</p>
				<p class="help">If you create mods yourself or know of other mods that you'd like to see here, let MarkByron know!</p>
